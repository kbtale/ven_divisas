# VenDiv

VenDiv is a currency monitor for the Venezuelan market. It tracks the exchange rate between the local Bolivar (VES) and international currencies: the US Dollar (USD), Euro (EUR), and Colombian Peso (COP).

## Features

- **Live Rates**: Real-time display for official (BCV) and parallel market rates.
- **Calculator**: Currency converter supporting VES, USD, EUR, and COP cross-rates.
- **History**: 30-day historical chart based on official records.
- **Design**: Fully responsive layout for mobile, tablet, and desktop devices.
- **Accessibility**: Implemented using semantic HTML landmarks and ARIA labels.

## Technology Stack

- **Core**: Vue 3 (Composition API) with Vite 5.
- **State Management**: Pinia for currency and application state.
- **Graphics**: Chart.js 4 for 30-day history visualization.
- **Styling**: Bootstrap 5 and custom CSS.
- **Data Source**: DolarApi.com.

## Project Status

While VenDiv was completely modernized in 2026 to restore full functionality and resolve legacy deprecations, it has not been actively used since 2021. As a result, this repository is now being archived.
